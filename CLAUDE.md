# CLAUDE.md

This file provides guidance to Claude Code (claude.ai/code) when working with code in this repository.

## Repository Purpose

Personal knowledge base for tech notes: conference talks, training sessions, article summaries, video notes, and ideas. All content is Markdown, no build system or tests.

## Structure

- `events/` — Conference and meetup notes (Devoxx, BDX IO, Staff42...)
- `trainings/` — Training session notes (DDD, OpenRewrite, TOGAF...)
- `articles/` — Article summaries (Tech Radar...)
- `videos/` — Video watch notes
- `assets/` — Images and diagrams referenced by notes
- `IDEAS.md` — Freeform thoughts that may become articles or experiments

## Conventions

- File naming: `YYYY-MM-DD_Event_Name.md` (or `YYYY-MM_Topic.md` for month-scoped notes)
- Notes are written in French or English depending on the source material
- `README.md` serves as the index — new notes should be linked there under the appropriate section
- Git branch model: `main` is the default branch
